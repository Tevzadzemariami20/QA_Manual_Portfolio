### Overview
This folder contains a collection of **manual bug reports** created during testing of web applications.  
The bugs are documented using industry-standard QA practices and clearly describe the issue, impact, and steps to reproduce.

### Each bug report includes:
- Clear title
- Test environment details
- Preconditions
- Steps to reproduce
- Expected result
- Actual result
- Severity and priority
- Reproducibility rate
- Supporting evidence (screenshots)

### Bug Report Structure
All bug reports follow this format:

- **Title**
- **Test Environment**

  - Browser
- **Preconditions**
- **Steps to Reproduce**
- **Actual Result**
- **Expected Result**
- **Severity / Priority**
- **Reproducibility Rate**
- **Status**
