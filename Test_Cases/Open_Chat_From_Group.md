# Verify Opening a Chat from a Group

### Priority: High  
### Type: Functional  
### Feature: Chat List & Chat Groups  
### Status: Pass  

## Description
- Verify that a user can successfully open a chat from a chat group and view the chat content.

## Pre-conditions
- User is registered
- User is logged in
- User has access to chat groups

## Post-conditions
- Selected chat is opened
- Chat messages are visible and readable

## Test Steps
| Step No. | Action | Input Data | Expected Result |
| -------- | ------ | ---------- | --------------- |
| 1 | Navigate to | The website opens successfully |
| 2 | Log in and navigate to the chat dashboard | Chat dashboard is displayed |
| 3 | Expand any chat group | The chat group expands and displays available chats |
| 4 | Click on any chat title within the group | The selected chat opens successfully and the chat content is visible and accessible |
